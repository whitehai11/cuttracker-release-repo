1. Xcode → Archive → Export signed IPA
2. GitHub Release mit Tag `vX.Y.Z` erstellen, `CutTracker.ipa` hochladen
3. In `apps.json` anpassen: `version`, `versionDate`, `downloadURL`, `size` (Bytes der IPA)
